Mesh Driven Cliffs

A Houdini system for procedurally generating rock cliff formations from any desired input mesh.

To be used in any game engine as a tool to ease level design and environment art.

2019

A procedural cliff generator prototype created using Houdini. The system takes any 3d mesh as input and generates a realistic rock cliff formation from a defined set of parameters to quickly generate believable assets.

Work in progress. Will become a multipurpose rock shaper from input mesh with several properties such as edge wear control, shattering and bricks generation.

The user can control the amount of rock layers and other erosion and generation parameters in a convenient interface.
